# N-Body-Coulomb-interaction-Animation

## Summary
This repository contains the Jupyter notebook used to create an animated visual of the N-Body problem applied to electrons and protons using the Coulomb force. The animation is fully numerically integrated using verlet integration, and the animation was createed using matplots animation capabilities. Particles are defined as objects using classes, initial positions are randomized, and initial velocities are given by the Maxwell–Boltzmann distribution, and number of protons and electrons is prompted in initializations. Result is a .gif file containing an animation of the specified number of protons and electrons interacting in a box, with accuratly updating positions and velocities.
